# DotFiles
## Requirements
1. Xorg
2. i3 package
3. zsh
4. polybar
5. rofi
6. xinit

## Usage
- While in terminal in Arch Linux
```zsh
# Clone this repository
git clone https://github.com/c043/c043-rice
```
- Move the contents of the config directory to $HOME/.config/
```zsh
cd ./c043-rice
mv -r ./config/* ../.config/
```
- Move the other dot files int the $HOME directory
```zsh
mv ./* ../
```
- Source .zshrc
```zsh
cd ..
source ./.zshrc
```
